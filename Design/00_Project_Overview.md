# Cat Ninja 3 — Project Overview

## Project Status
Preproduction / design planning

## Working Title
Cat Ninja 3

## Genre
Single-player 2D precision action platformer

## Platform
Desktop web browser

## Deployment Target
Vercel

## Core Input Scheme
- Keyboard for movement and actions
- Mouse for aiming and interaction

## High Concept
Cat Ninja 3 is a browser-based HD-2D pixel-art precision action platformer starring a mostly realistic house cat in ninja gear. The game takes place in exaggerated domestic and urban-commercial spaces such as apartments, streets, malls, and office interiors, reimagined at cat scale as dramatic traversal and combat arenas.

The player uses heavy, deliberate movement, double jump, wall jump, a damaging dash, limited-ammo mouse-aimed shuriken, a precise claw strike, a manually activated catnip frenzy mode, and a time-charged hairball super to defeat absurd enemies including combat-skilled grandmothers and a laser-shooting T-Rex miniboss.

## Core Fantasy
Be a house cat turned action hero: fast, stylish, precise, dangerous, and slightly ridiculous.

## Tone
- Playful retro parody
- Cool arcade action
- Absurd but readable
- Funny without becoming random nonsense

## Primary Design Pillars
1. **Precision with weight**  
   Movement and combat should feel committed, physical, and timing-based.

2. **House-cat action fantasy**  
   The hero should feel like a real house cat elevated into ninja-action absurdity.

3. **Stylish absurdity**  
   The world is funny and exaggerated, but the gameplay remains crisp and readable.

4. **Fast arcade recovery**  
   Short death loops and a per-level lives system keep momentum high.

## MVP Goal
Deliver a polished vertical slice that proves:
- movement feels good
- mouse-aimed shuriken works in a platformer
- the combat loop is readable and satisfying
- the world/theme is funny and distinctive
- a first boss encounter can be completed in a single short level

## MVP Content Summary
- 1 playable character
- 1 level
- 2 basic enemy types
- 1 miniboss
- health bar + 9 lives per level
- shuriken ammo pickups
- catfood pickups
- catnip pickups
- frenzy mode
- hairball super
- mall atrium boss finale

## Current Engine Recommendation
- Phaser 3
- Vite
- TypeScript
